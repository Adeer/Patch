
О продавцах:
Вендоры очень удобны - все итемы разделены по группам ( только для 0-29 лвл , для 20-59 , для 60-69 ,для 70 )
оружия разделены по классам (только луки ,только одноручные мечи , только двуручные мечи  и тд )
броня разделена по классам ( только легкие :cloth ,leather ,только тяжелые : mail,plate )
Наполнение  вендоров максимально :например количество оружия в вендорах составляет ~2.7 тысяч штук  ,примерно столько же покажет  и  wowhead.com 
Множество сетов брони: Тир 1-6 , Арена 1-4 сезоны ,Данжеон 1-3 , рар и эпик сеты для 60 и 70 левелов 
Кроме оружия и брони есть множество других видов придметов :еда и напитки , зелья ,элексиры ,фласки , свитки ,энчант свитки , все виды сокет-камней ,стрелы и пули , и многое другое
Всего 256 вендора , которые продают 20717  разнообразных вещей 


Файлы:
AlterShop.sql - сами вендоры
item_FIX.sql - исправления квестового оружия и изменения цен для высоколевельных серверов.
Add quest and money,sql - Сам NPC С квестом дающий 70 лвл 5000 голда
ID вендоров оружия.txt- номера продавцов оружия для расстановки
ID вендоров брони.txt- номера продавцов брони для расстановки
ID вендоров остальной брони.txt-номера остальной брони для расстановки
ID вендоров сетов.txt-номера продавцов сетов
ID вендоров разного.txt- номера продавцов других вещей
история версий.txt- информация о изменениях в  версиях вендоров
папка spawn- частичная расстановка продавцов ( автор: ATLANT с форума mangos.ru )
папка Multiplier Price- файлы для увеличения цен 

порядок применения файлов , устанавливать в базу мира(mangos по умолчанию):
1 AlterShop.sql
2 item_FIX.sql
